# Marketing Mix Modeling (MMM)
## Overview

This project uses Marketing Mix Modeling (MMM) to quantify the impact of different marketing channels on conversions and revenue, and to optimize budget allocation.

## Objective
Measure contribution of each channel (e.g., paid search, SEO, social, email)
Understand ROI and diminishing returns
Provide data-driven budget recommendations
## Methodology
Aggregated and cleaned time-series marketing and performance data
Engineered features (lag effects, adstock, seasonality)
Built regression-based models to estimate channel impact
## Results
Identified highest ROI channels and diminishing returns curves
Quantified channel contributions to total conversions
Generated optimized budget allocation scenarios
## Tech Stack
Python (pandas, statsmodels, scikit-learn)
SQL
Tableau / Power BI
## Key Takeaway
MMM provides a holistic, data-driven view of marketing performance, enabling more efficient budget allocation across channels.
